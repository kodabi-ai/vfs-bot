## 🔄 2. Ana Otomasyon Akışları

### 🌐 Flow 1: Login & Cookie Bypass (V50 Stratejisi)
1. **Cloudscraper Bypass:** `delay=10` ile başlangıç, cookie topla, domain `.vfsglobal.com` olarak enjekte et.
2. **Root Init:** Önce ana portal (`/`) aç, Angular shell'i yükle.
3. **Target Navigate:** `https://visa.vfsglobal.com/tur/tr/fra/login` rotasına git.
4. **CF Iframe Detach:** `iframe[src*='challenges/']` state `detached` olana kadar bekle (max 45s).
5. **Content-Await (V50):** `document.body.innerText.includes('E-posta')` true olana kadar bekle. Bu, SPA'nın form render'ının tamamlanmasını garanti eder.
6. **Selector & Fill:** Form elemanları bulunduktan sonra credentials (`mustafa.eke@live.com` / `Vfsglobal!5561!`) gir.
7. **Submit:** `button:has-text("Giriş Yap")` veya `type="submit"` tıkla.
8. **API Trigger:** `POST /api/v1/auth/otp` çağrısı yapılır, `200 OK` döner.

### 📧 Flow 2: OTP Yakalama & Dashboard
- **Primary:** Email IMAP polling (`mustafa.eke@live.com`). Regex: `\d{6}`.
- **Backup:** SMS `5468224662`. API veya webhook kontrolü.
- **Dashboard:** OTP doğrulandıktan sonra `fra/dashboard` rotasına yönlendirilir. API endpoints kontrol edilir.
- **Yeni Mimari:** `VFSOTPManager` sınıfı içinde `trigger_otp()` metodu ile otomatik tetikleme ve label-based input eşleşmeleri uygulanır.

- Versiyon sayıları (`v50`, `v49`) dosya adlarından silinip, mantıksal sınıflara (`VFSLoginBot`, `VFSOTPManager`) dönüştürüldü.
- `config.py` ile tüm sabitler, timeout'lar ve selector'lar merkezi hale getirildi.
- Kod tekrarı azaltıldı, `Single Responsibility Principle` (SRP) uygulandı.
- Legacy dosyalar `src/legacy/` klasörüne taşındı.

## 🛠️ 4. Teknik Best Practices & Konfigürasyon

### 🔐 Kimlik Bilgileri & Env
- `EMAIL`: `mustafa.eke@live.com`
- `PASS`: `Vfsglobal!5561!`
- `PHONE`: `5468224662`
- `VFS_HOST`: `https://visa.vfsglobal.com/tur/tr/fra/login`
- `API_BASE`: `https://lift-api.vfsglobal.com`

### 🕵️‍♂️ Debug & Hata Çözümleme
1. **İframe Takılması:** `await page.wait_for_selector("iframe[src*='challenges/']", state="detached", timeout=45000)`
2. **Form Bulunamaması:** `document.body.innerText.includes('E-posta')` kontrolü yap. Eğer false ise sayfayı `reload(wait_until="load")` yap.
3. **DNS/Network:** Container içinden `host.docker.internal:11434` (Ollama) veya `lift-api.vfsglobal.com` timeout alıyorsa `/etc/hosts`'a statik IP ekle.
4. **HTML Dökümü:** `page.content()` veya `page.inner_text('body')` log'la. Formun `vfs-app` veya `ng-content` içinde render olup olmadığını kontrol et.
5. **Screenshots:** Başarısız durumlarda `screenshots/e2e_vXX_debug.png` otomatik kaydedilir. DOM görseli incele.

### 📦 Cache & Redis (Phase 6)
- Redis bağlantısı `redis.ConnectionPool.from_url()` ile yönetilir.
- `max_connections=10`, `decode_responses=True`.
- Hata durumunda `MemoryFallback.get(key)` tetiklenir.
- TTL: 3600s.
